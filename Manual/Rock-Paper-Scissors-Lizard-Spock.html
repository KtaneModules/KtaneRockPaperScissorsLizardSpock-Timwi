On the Subject of Rock-Paper-Scissors-Lizard-Spock

Anecdotal evidence suggests that in the game of Rock-Paper-Scissors, players familiar with each other will tie 75 to 80% of the time due to the limited number of outcomes. Rock-Paper-Scissors-Lizard-Spock was created by Internet pioneer Sam Kass as an improvement on the classic game. All hail Sam Kass. Hail.

The rules are very simple. Scissors cuts paper. Paper covers rock. Rock crushes lizard. Lizard poisons Spock. Spock smashes scissors. Scissors decapitates lizard. Lizard eats paper. Paper disproves Spock. Spock vaporizes rock. And, as it always has, rock crushes scissors.

Determine which icon is presented by the bomb, then click the two icons that beat it in any order.

Condition Bomb’s choice
If there is an RJ-45 port, Rock.
Otherwise, if there is a parallel and a serial port, Spock.
Otherwise, if there is a parallel port, Paper.
Otherwise, if there is a serial port, Scissors.
Otherwise, if there is any lit indicator, Lizard.
Otherwise, if there is a Stereo RCA port, Rock.
Otherwise, if there is a PS/2 port, Paper.
Otherwise, if there is an unlit SND indicator, Spock.
Otherwise, if there is an unlit SIG indicator, Scissors.
Otherwise, if there is any other unlit indicator, Lizard.
Otherwise, if the serial number contains a vowel, Rock.
Otherwise, if there is a DVI-D port, Paper.
Otherwise, if there is an even number of batteries, Scissors.
Otherwise, if there is a Type AA battery, Lizard.
Otherwise, Spock.
